Aims

This unit is designed to introduce the technology of the Internet and World Wide Web and to develop an understanding of the technologies associated with programming for the World Wide Web.

Learning Objectives

After successfully completing this unit, you should be able to:

  1. Develop web pages using HTML, JavaScript and CSS.
  2. Demonstrate an understanding of the key technologies of the World Wide Web.
  3. Describe the protocols of the World Wide Web.
  4. Design effective and user-friendly web pages using usability principles and style guides.
  5. Demonstrate an understanding of the history of the Internet and World Wide Web.

Content

  1. Web concepts: definition, history and fundamental concepts
  2. Introduction to HTML, CSS and Web Development processes
  3. HTML5: logical document structure, syntax, images, links, maps, tables, frames, forms
  4. W3C standards, validation
  5. Design, Layout and Usability: principles of navigation, usability, page style guides, standards
  6. Protocols and server technology: HTTP, TCP/IP, MIME, URI, DNS
  7. JavaScript: syntax, object model, event handling, common tasks
  8. Interactivity: DOM, and some evolving technologies forms processing, common tasks
  9. Style sheets: fundamentals, CSS formatting, CSS positioning, standards
  10. Introduction to XML:
    1. Syntax
    2. DTDs
    3. XSL
    4. XML Applications
  11. Web design and usability: principles of navigation, usability, style guides, standards
